# GoPoint

[Latest Release](https://github.com/ruruoni1/GoPoint/releases/latest)

GoPoint is a Windows mouse trail overlay for tutorials, presentations, streaming, and screen sharing.

## Official Website

- Main landing page: [GoPoint Pages](https://ruruoni1.github.io/GoPoint/)
- English page: [GoPoint English](https://ruruoni1.github.io/GoPoint/en/)
- Korean page: [GoPoint Korean](https://ruruoni1.github.io/GoPoint/ko/)

## Highlights

- Smooth spring-based cursor tracking
- Constant, Dots, and Tapered trail styles
- Multi-color gradients and adjustable width/length/fade
- System tray control, Windows startup, and auto-update
- Low-spec mode for older PCs
- Duplicate launch prevention in the tray workflow

## Download

Download the latest `GoPoint.exe` from:

## Notes

- Users on `v1.0.12` need one manual install because auto-update detection was broken in that release.
- Local auto-update testing is supported through `update-test/update.json` or `GOPOINT_UPDATE_MANIFEST`.
