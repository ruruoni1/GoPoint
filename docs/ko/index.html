GoPoint
한국어 English

Windows Mouse Trail Overlay

부드럽고 가벼운 마우스 트레일

GoPoint는 튜토리얼 영상, 프레젠테이션, 방송, 화면 공유 중에 마우스 위치를 더 또렷하게 보여주는 Windows 전용 오버레이 유틸리티입니다. 스프링 기반 추적과 단계형 저사양 모드로 부드러움과 가벼움을 같이 잡았습니다.

최신 다운로드 GitHub 저장소
최신 버전 v1.0.16
핵심 방향 가볍고 자연스러운 추적
배포 방식 단일 EXE + 자동 업데이트
중요 v1.0.12 사용자는 자동 업데이트 판별 오류 때문에 이번 한 번은 수동 설치가 필요합니다.

주요 기능

  • Constant, Dots, Tapered 스타일 지원
  • 다중 색상 그라데이션과 폭, 길이, 투명도 조절
  • 시스템 트레이, 시작프로그램 등록, 자동 업데이트
  • 저사양 모드 단계 조절
  • 중복 실행 방지

v1.0.16 변경

  • 트레이 실행 중 EXE를 다시 열면 기존 설정창만 다시 표시
  • Nuitka onefile 시작프로그램 경로 보정 유지
  • GitHub 없이 로컬 자동 업데이트 테스트 지원
  • 기존 성능 최적화와 저사양 모드 유지

Performance

오래된 PC에서도 가벼운 동작을 유지

프레임 갱신과 미리보기 타이머를 최적화해 오래된 PC 환경에서도 CPU 점유를 낮추는 방향으로 정리했습니다.
