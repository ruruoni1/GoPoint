Windows Mouse Trail Overlay

Smooth, readable cursor trails for live demos.

GoPoint is a Windows overlay utility for tutorials, presentations, streaming, and screen sharing. It combines spring-based tracking, lightweight rendering, and a single-file EXE workflow that stays practical for everyday use.

Highlights

  • Gradient colors, width, length, and fade controls

  • Tiered low-spec mode for older PCs

What changed in v1.0.16

  • Launching the EXE again now reopens the existing tray instance instead of starting another copy
  • Nuitka onefile startup path handling stays pinned to the real GoPoint.exe path
  • Local auto-update testing now works through update-test/update.json
  • Performance optimizations from v1.0.13 remain intact

Performance

Still tuned for older hardware

The rendering path, preview timer behavior, and low-spec presets were kept focused on older PCs so GoPoint stays usable as an always-on utility.
